# Final Steps to Start the Backend

Follow these steps exactly to get the backend running:

1.  **Open MySQL Workbench** (or your MySQL command line) and run:
    ```sql
    CREATE DATABASE agrigov_db;
    ```
    *(If it already exists, skip this step)*

2.  **Open a NEW Terminal** in this folder.

3.  **Move to the Backend folder**:
    ```powershell
    cd "C:\Users\Informatics\Desktop\projet memoire\resources\Backend"
    ```

4.  **Create the tables**:
    ```powershell
    python manage.py migrate
    ```

5.  **Start the server**:
    ```powershell
    python manage.py runserver
    ```
